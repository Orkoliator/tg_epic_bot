# Epic Telegram Bot

This bot is made to get information about weekly free games and share it in telegram chat.

## Configuration
It is necessary to set variables for your telegram bot. There are two ways to do it:
1) using var.yaml file
2) using system environmet

Mandatory variables are:
- TG_API_ID
- TG_API_HASH
- TG_BOT_TOKEN

### setting variables with file
To configure bot this way just rename "var_example.yaml" file to "var.yaml" and set mandatory variables in it.

### setting variables with env
To configure this way set mandatory variables as system environment with same names.
